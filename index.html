Current Projects

EFlame Engine

A simple OpenGL-based game engine that I am developing as a foundational framework, with the primary goal of using it to practice and experiment with various concepts in computer physics, including collision detection, rigid body dynamics, and real-time simulation.

SkyBlock Recreation

The best commercial Hypixel SkyBlock Sandbox / Recreation currently available. Contains Gemstones, 70% of Hypixels Items, Dwarven Mines, Guilds, Auction House, NPC shops, Quests, Islands, Bazaar and just about everything else!

Cora

My experiments with the Vulkan Graphics API using the C programming language, where I explore low-level graphics programming concepts, rendering techniques, and performance optimization.

Atlas Network

My largest and longest-running project to date, and one of, if not the best, active recreations of Hypixel Skyblock, featuring a thriving community of over 9,000 members. The project is constantly evolving with regular updates, new features, and a passionate player base that continues to grow.

Dnum Framework

A lightweight and modular framework featuring a suite of pre-built libraries including Commands, NPCs, Holograms, SignGUI, Packet Listener, and GUI components. Available on GitHub for exploration and contribution.

Reimagined Engine

A graphics engine in development—soon evolving into a full game engine—built using Microsoft DirectX and OpenGL. It includes core rendering features and a semi-functional editor UI. Explore the full project on GitHub for more details.
